# MusicX v1.6.1

## Features
- Widget support: MiniPlayer, MediumPlayer, LargeAlbum, Favorites, PlaylistLauncher, QuickActions, Shuffle widgets
- In-app update system with DownloadManager integration
- YouTube import improvements with better error messages
- Import flow enhancements for playlist and song sources
- Playback performance improvements
- Build config changes for version management

## Changes
- Upgraded to Glance 1.1.1 for widget rendering
- Added UpdateReleaseParser for fetching release info from GitHub
- Improved settings screen with update UI
- Various bug fixes and optimizations
