##  About

MusicX is a fast, minimal offline music player built with a clean black-and-white design.

I started building MusicX because almost every music app I tried was filled with subscriptions, locked features, ads, and paywalls everywhere — especially for customization and basic features. I wanted a music player that felt simple, smooth, fully customizable, and actually focused on the user experience instead of pushing premium plans.

That’s why MusicX is completely free to use, with no paid plans or locked customization features. The only monetization planned is optional donations for people who want to support development.

MusicX is designed to be:
- Fast
- Minimal
- Offline-first
- Fully customizable
- Clean and distraction-free

### Features
-  Offline playback
-  Local music library
-  Theme customization
-  Smooth performance
-  Animated splash screen *(planned)*
